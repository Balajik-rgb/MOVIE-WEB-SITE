# MOVIE-WEB-SITE
🎬 Sri Balaji Movie Zone — Cinematic Movie Website Experience 🍿🔥

Welcome to Sri Balaji Movie Zone, a futuristic movie-inspired web experience designed with premium UI aesthetics, cinematic animations, glowing visuals, and immersive frontend interactions.

This project combines creativity, animation, and responsive design into a modern entertainment-style interface built completely using HTML, CSS, and JavaScript.

✨ Features
🎥 Dynamic Hero Movie Slider
🌌 Animated Floating Particle Background
🔥 Premium Neon Glow UI Effects
⚡ Smooth Hover & Transition Animations
🔍 Interactive Search Bar
📱 Fully Responsive Layout
🎨 Cinematic Dark Theme Design
🚀 Fast & Optimized Frontend Performance
💡 Modern Navigation & Visual Effects
🛠 Technologies Used
HTML5
CSS3
JavaScript
Google Fonts
Advanced CSS Animations
🚀 Project Highlights

This project was created to explore advanced frontend development concepts including:

✔ Modern UI/UX Design
✔ Responsive Web Development
✔ Creative Animation Systems
✔ Interactive User Experience
✔ Structured & Clean Code Architecture

The design is inspired by modern streaming platforms while maintaining a unique futuristic cinematic identity.
